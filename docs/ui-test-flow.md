## 交互测试步骤

### 测试 1: 紧凑模式切换

```
初始状态: ☑ Compact mode
输出: "We're no strangers to love. You know the rules and so do I."

点击后: ☐ Compact mode
输出: "[00:00] We're no strangers to love
       [00:05] You know the rules and so do I"
```

### 测试 2: 时间戳切换

```
初始状态: ☐ Include timestamps
输出: "We're no strangers to love..."

点击后: ☑ Include timestamps
输出: "[00:00](url) We're no strangers to love..."
```

### 测试 3: 组合测试

```
组合 1: ☑ Compact + ☐ Timestamp
→ 纯文本段落

组合 2: ☑ Compact + ☑ Timestamp
→ 段落带开头时间戳

组合 3: ☐ Compact + ☑ Timestamp
→ 每行带时间戳

组合 4: ☐ Compact + ☐ Timestamp
→ 纯原始片段
```

## 预期结果验证

| 场景 | 预期行为 | 实际结果 |
|------|----------|----------|
| 默认状态 | 紧凑模式开启 | ✓ |
| 切换紧凑模式 | Markdown 立即更新 | ✓ |
| 切换时间戳 | 时间戳出现/消失 | ✓ |
| 紧凑模式 OFF | 行数增加 | ✓ |
| 紧凑模式 ON | 行数减少 | ✓ |
| 复制按钮 | 复制当前格式内容 | ✓ |
| 下载按钮 | 下载当前格式 .md | ✓ |
