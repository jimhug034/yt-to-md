## 技术说明

### 客户端抓取机制
- 函数: `getTracksFromEmbed()` in `app/lib/youtube.ts`
- 原理: 从 YouTube embed 页面提取 `ytInitialPlayerResponse`
- 优势: 不依赖第三方服务，更可靠
- 限制: 需要能访问 YouTube

### API 降级机制
- 路由: `/api/subtitles`
- 原理: 使用 Invidious 实例作为代理
- 降级: 返回常用语言列表
- 限制: 公共实例经常不可用
